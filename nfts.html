Lesson 3: NFTs in GameFi

Non-Fungible Tokens (NFTs)

NFTs allow players to truly own game assets. This lesson explains how NFTs work in GameFi and how to create your own NFTs for in-game items.

Minting NFTs

pragma solidity ^0.8.0;

contract GameNFT {
    mapping(uint => address) public owners;
    uint public totalSupply;

    function mintNFT() public {
        totalSupply++;
        owners[totalSupply] = msg.sender;
    }
}
