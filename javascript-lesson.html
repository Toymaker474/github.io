Lesson 4: JavaScript Basics

What is JavaScript?

JavaScript is a powerful programming language that allows you to add interactivity to your website. In this lesson, we will explore the basics.

Example: JavaScript Variables

// Declare a variable
var myName = "Tyler";

// Display it in the console
console.log(myName);

Try running the code above in your browser's developer console to see the result!
